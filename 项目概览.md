## 🎯 项目简介

将传统佛学网站（http://www.cfolu.com/）重构为现代化、简约、便捷的在线学习平台。

### 核心特点
- ✅ 纯静态网站，无需服务器
- ✅ 全站内容搜索（客户端索引）
- ✅ 保留原有文章颜色样式
- ✅ 响应式设计，支持手机/平板
- ✅ 支持多板块（净修院、禅修院、修学园地）
- ✅ 本地管理，GitHub 托管

## 🎨 原网站架构

### 三大板块

```
如说修行网上佛学院
│
├── 净修院 (index.htm)
│   ├── 目标：念佛往生，一心不乱
│   ├── 学习方式：四个学期课程体系
│   └── 课程内容：
│       ├── 第一学期：培养目标（地藏经、了凡四训等）
│       ├── 第二学期：以戒为师（四十二章经、八大人觉经等）
│       ├── 第三学期：解行并进（金刚经、心经、六祖坛经等）
│       └── 第四学期：以证为归（称赞净土经、念佛论等）
│
├── 禅修院 (index-chan.htm)
│   ├── 目标：明心见性，顿悟成佛
│   ├── 学习方式：四个学期课程体系
│   └── 课程内容：
│       ├── 第一学期：培养目标
│       ├── 第二学期：以戒为师
│       ├── 第三学期：解行并进
│       └── 第四学期：见性等持（二入四行论、修心诀等）
│
└── 修学园地 (xiuxueyd/index.htm)
    ├── 目标：实修实证，解行并进
    ├── 特点：大量修行指导文章（290+ 篇）
    └── 内容：
        ├── 自在学佛
        ├── 本愿法门系列
        ├── 善导大师系列
        ├── 禅修系列
        ├── 念佛指导
        └── 经典注解
```

### 导航特点
- 三个板块相互独立但互联
- 每个板块都有去其他两个板块的链接
- 形成三角形互联结构

## 💻 技术栈

### 前端
- **框架**: 原生 HTML/CSS/JavaScript（无框架依赖）

## 🚀 使用流程

### 开发阶段

1. **解析内容**
```bash
cd webapp
pip install beautifulsoup4
python parser.py
```

2. **本地预览**
```bash
# Windows
start.bat

# macOS/Linux
python -m http.server 8000
```

3. **访问**
```
http://localhost:8000/index-new.html  # 新版
http://localhost:8000/index.html      # 旧版
http://localhost:8000/compare.html    # 对比
```

### 部署阶段

1. **推送到 GitHub**
```bash
git init
git add .
git commit -m "初始化项目"
git remote add origin https://github.com/用户名/仓库名.git
git push -u origin main
```

2. **启用 GitHub Pages**
- Settings → Pages
- Source: main branch
- 5 分钟后访问

3. **访问地址**
```
https://用户名.github.io/仓库名/
```

### 维护阶段

1. **修改内容**
```bash
# 编辑 My Web Sites/ 中的 HTML 文件
# 重新运行解析器
python parser.py

# 提交更新
git add .
git commit -m "更新文章"
git push
```

2. **自动部署**
- GitHub Pages 自动部署
- 1-5 分钟生效

## 🔍 常见任务

### 添加新文章
1. 在 `My Web Sites/` 添加 HTML 文件
2. 运行 `python parser.py`
3. 提交并推送

### 修改样式
1. 编辑 `index-new.html` 中的 Tailwind 类
2. 或修改内联样式
3. 提交推送

### 添加新分类
```javascript
// 在 app-new.js 中修改
function getCategoryFromFile(file) {
    if (file.includes('dao')) return '道学';
    if (file.includes('zhe')) return '哲学';
    // 添加新分类
    if (file.includes('new')) return '新分类';
    return '佛学';
}
```
