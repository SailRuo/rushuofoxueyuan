## 🚀 快速预览

### 方法一：使用启动脚本（Windows）

```bash
# 双击运行
preview-redesign.bat

# 或在命令行运行
cd webapp
preview-redesign.bat
```

然后在浏览器访问：`http://localhost:8000/index-redesign.html`

### 方法二：手动启动

```bash
# 进入目录
cd webapp

# 启动 Python 服务器
python -m http.server 8000

# 在浏览器访问
http://localhost:8000/index-redesign.html
```

### 方法三：查看对比

```bash
# 启动服务器后访问对比页面
http://localhost:8000/compare-redesign.html
```

## 🔧 开发指南

### 修改颜色

编辑 `index-redesign.html` 中的 CSS 变量：

```css
:root {
    --color-primary: #4F46E5;      /* 主色 */
    --color-secondary: #818CF8;    /* 次要色 */
    --color-cta: #22C55E;          /* 强调色 */
    --color-background: #EEF2FF;   /* 背景色 */
    --color-text: #312E81;         /* 文字色 */
}
```

### 添加新板块

1. 在 `courseData` 中添加数据：

```javascript
const courseData = {
    // ... 现有板块
    newSection: {
        name: '新板块',
        goal: '目标描述',
        color: '#059669',
        semesters: [...]
    }
};
```

2. 在 HTML 中添加导航标签：

```html
<button class="nav-tab" data-section="newSection">新板块</button>
```

3. 添加内容区域：

```html
<section id="newSection" class="section-view">
    <!-- 内容 -->
</section>
```

### 修改课程数据

编辑 `app-redesign.js` 中的 `courseData` 对象：

```javascript
courses: [
    { title: '课程名称', file: '文件名.html' },
    // 添加更多课程...
]
```

## 🐛 常见问题

### Q: 页面空白？

**A**: 检查以下几点：
1. 确认 `articles.json` 存在
2. 检查浏览器控制台错误
3. 确认服务器正在运行
4. 清除浏览器缓存

### Q: 搜索不工作？

**A**: 
1. 确认 `articles.json` 加载成功
2. 检查控制台是否有错误
3. 尝试刷新页面

### Q: 样式错乱？

**A**:
1. 清除浏览器缓存
2. 确认 CSS 正确加载
3. 检查是否有 CSS 冲突

### Q: 文章详情不显示？

**A**:
1. 确认 `articles.json` 格式正确
2. 检查文章数据是否包含 `paragraphs`
3. 查看控制台错误信息
