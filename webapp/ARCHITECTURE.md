# 如说修行 - 网站架构分析

## 原网站结构

### 三大板块

```
如说修行网上佛学院
├── 净修院 (index.htm)
│   ├── 目标：念佛往生，一心不乱
│   ├── 四个学期课程
│   └── 主要内容：净土宗经典
│
├── 禅修院 (index-chan.htm)
│   ├── 目标：明心见性，顿悟成佛
│   ├── 四个学期课程
│   └── 主要内容：禅宗经典
│
└── 修学园地 (xiuxueyd/index.htm)
    ├── 目标：实修实证
    ├── 290+ 篇文章
    └── 主要内容：修行指导、经典注解
```

### 课程体系

每个院都有四个学期：
1. 一、培养目标
2. 二、以戒为师
3. 三、解行并进
4. 四、以证为归（净修院）/ 四、见性等持（禅修院）

### 内容特点

- **净修院**：专注净土法门，念佛往生
- **禅修院**：专注禅宗法门，明心见性
- **修学园地**：综合修学资料，实修指导

## 重新设计方案

### 方案一：统一入口 + 三个子站

```
主页 (index.html)
├── 选择板块
│   ├── 进入净修院 → jingxiu/index.html
│   ├── 进入禅修院 → chanxiu/index.html
│   └── 进入修学园地 → xiuxueyd/index.html
│
└── 全站搜索（跨三个板块）
```

### 方案二：单页应用 + 标签切换

```
主页 (index.html)
├── 顶部导航：[净修院] [禅修院] [修学园地]
├── 搜索框（全站）
└── 内容区（动态切换）
    ├── 净修院内容
    ├── 禅修院内容
    └── 修学园地内容
```

### 方案三：保留原结构 + 现代化界面

```
保留三个独立入口
├── index.html（净修院）- 重新设计
├── index-chan.html（禅修院）- 重新设计
├── xiuxueyd/index.html（修学园地）- 重新设计
└── portal.html（新增：统一门户）
```

## 推荐方案

**方案二：单页应用 + 标签切换**

### 优势
- ✓ 统一的用户体验
- ✓ 全站搜索更方便
- ✓ 无需页面跳转
- ✓ 保留三个板块的独立性
- ✓ 移动端友好
