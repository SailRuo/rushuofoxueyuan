## 设计特点

### 1. 三个独立板块

| 板块 | 目标 | 特点 | 文章数 |
|------|------|------|--------|
| **净修院** | 念佛往生，一心不乱 | 系统课程，4学期 | ~30 篇 |
| **禅修院** | 明心见性，顿悟成佛 | 系统课程，4学期 | ~30 篇 |
| **修学园地** | 实修实证 | 综合资料库 | 290+ 篇 |

### 2. 课程体系

每个院都有四个学期的递进式学习：

**净修院四学期：**
1. 培养目标 - 建立正确知见
2. 以戒为师 - 持戒修行
3. 解行并进 - 理论与实践结合
4. 以证为归 - 念佛往生

**禅修院四学期：**
1. 培养目标 - 发菩提心
2. 以戒为师 - 戒定慧
3. 解行并进 - 禅修实践
4. 见性等持 - 明心见性

### 3. 导航特点

- 每个板块都有去其他两个板块的链接
- 形成三角形互联结构
- 用户可以自由切换学习内容

### 4. 内容特点

**净修院 & 禅修院：**
- 结构化课程
- 表格形式展示
- 每学期 5-7 门课程
- 每日学习建议

**修学园地：**
- 资料库形式
- 5列表格布局
- 按主题分类
- 包含音频、PDF、HTML

## 用户学习路径

### 路径一：系统学习（净修院/禅修院）
```
入口 → 选择板块 → 第一学期 → 第二学期 → 第三学期 → 第四学期
```

### 路径二：主题学习（修学园地）
```
入口 → 修学园地 → 按主题浏览 → 选择文章阅读
```

### 路径三：交叉学习
```
净修院 ←→ 禅修院 ←→ 修学园地
（三个板块自由切换）
```

## 设计理念

1. **系统性** - 净修院和禅修院提供系统化课程
2. **实用性** - 修学园地提供实修指导
3. **互补性** - 三个板块相互补充
4. **灵活性** - 用户可自由选择学习路径
5. **传承性** - 保留传统佛学教育体系

## 技术特点

- **纯静态** - 无需服务器
- **离线可用** - 可下载离线阅读
- **表格布局** - 传统 HTML 表格
- **内联样式** - 颜色直接写在 HTML 中
- **GB2312 编码** - 简体中文
- **目标新窗口** - `<base target="_blank">`
